**📱 TaskMate Wireframe**

This project presents low-fidelity wireframes for a Task Management mobile application **"TaskMate"** designed as part of a UI/UX internship task.

**🎯 Objective** : To design a simple and user-friendly task management app by focusing on layout, structure, and user flow.
